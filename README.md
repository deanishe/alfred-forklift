ForkLift Favourites for Alfred
==============================

Browse, filter and open [ForkLift 3][forklift] favourites from [Alfred][alfred].


Usage
-----

- `ftp [<query>]` — List/filter ForkLift favourites
    - `↩` — Open favourite in ForkLift
